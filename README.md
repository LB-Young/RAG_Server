# RAG_Server
A ui server about RAG system.

# Support Files Types
- .txt
- .pdf
- .docx

# Support LLM Saas API
- Deepseek 

# Start
```bash
pip install -r requirements.txt
python3 app.py
```
- 访问：http://localhost:5000/
